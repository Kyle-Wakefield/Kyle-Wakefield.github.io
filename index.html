Pokemon Teambuilder

Pokemon Teambuilder is a full stack application that allows users to build a team of up to 6 customized Pokemon. users can customize the species, nickname, moves, and ability of each Pokemon on the team, as well as the team's name. The back end is built with Express, MongoDB, and Mongoose. The front end is built with ReactJS and Bootstrap. The app also uses the 3rd-party PokeAPI.

FaceSpace

FaceSpace is a full-stack social media application that allows users to create their own posts and view all users' posts. I colaborated on this project with two other developers, and we used GitHub to organize and combine our code. The back end is built with Express, MongoDB, and Mongoose. The front end is built with React and React Bootstrap.

McDennis Review Board

McDennis Review Board is a full-stack restaurant review application for a fictional fast food joint. It allows users to create reviews, and view all users' reviews. Users can also filter reviews by item, or view just their own reviews. The back end is built with Express, MongoDB, and Mongoose. The front end is built with HTML, Bootstrap, and jQuery.

Tic-Tac-Toe

Tic-Tac-Toe is a front-end application that allows users to play Tic-Tac-Toe against themselves. The app is built with HTML, Bootstrap, and jQuery.
